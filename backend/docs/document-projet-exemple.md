# Document Exemple Projet

## Titre
Plateforme de mise en relation entre clients et travailleurs

## Description
Cette application permet aux clients de publier des taches et de reserver des prestations aupres de travailleurs verifies.  
Le backend gere l'authentification, les utilisateurs, les travailleurs, les taches, les reservations, les notifications et les evaluations.

## Objectifs
- Permettre a un client de creer un compte et se connecter par numero de telephone.
- Permettre a un travailleur de completer son profil et d'ajouter ses documents.
- Gerer les reservations entre clients et travailleurs.
- Envoyer des notifications lors des actions importantes.
- Donner la possibilite de noter un travailleur apres une prestation.

## Fonctionnalites principales
1. Authentification avec verification.
2. Gestion des profils utilisateur et travailleur.
3. Publication et suivi des taches.
4. Reservation d'un service.
5. Systeme de notation.
6. Notifications en temps reel.

## Technologies
- Java
- Spring Boot
- Spring Security
- JWT
- MySQL
- Maven

## Exemple de cas d'utilisation
Un client cree une tache pour demander un electricien.  
Un travailleur consulte la tache et accepte la demande.  
Le client confirme la reservation puis laisse une evaluation a la fin du service.
